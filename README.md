\# 📚 Multimodal RAG System



Локальная система для умного поиска по техническим книгам с поддержкой текста и изображений.



\## ✨ Возможности



\- 📖 Чтение PDF-книг любого размера

\- 🖼️ Распознавание и описание изображений, графиков, схем

\- 🤖 Локальный AI (Gemma 3 27B + LLaVA) — ничего не уходит в облако

\- 🔍 Поиск по смыслу, а не по ключевым словам

\- 🌐 Удобный веб-интерфейс

\- 📊 Показывает источники ответов (страницы, картинки)



\## 🛠️ Технологии



\- Python 3.10

\- LangChain

\- Ollama (Gemma 3 27B, LLaVA)

\- ChromaDB

\- FastAPI

\- Tesseract OCR

\- Poppler
